# Что такое WYSIWYG?

WYSIWYG расшифровывается как **«What You See Is What You Get»**.<br/>Это значит, что редактор показывает текст в том виде, в каком он будет выглядеть при публикации. Поэтому WYSIWYG редакторы интуитивно понятны и удобны для пользователей.

# Особенности редакторов

* IDE предназначены для работы с кодом;
* простые блокноты не подходят для сложных задач;
* WYSIWYG и редакторы кода с превью подходят<br/>для работы с контентом.

(end) ####################################
-->

# Редакторы для работы<br/>с контентом

* WYSIWYG: результат виден сразу, без предварительного просмотра, удобны для нетехнических пользователей.
* Редакторы кода с превью: разделяют редактирование и визуализацию, дают точный контроль над структурой и стилем.
