# Почему хотелось WYSIWYG?

* при работе с базой знаний хочется большей интерактивности;
* разметка бывает слишком многословной (например, ссылки и таблицы);
* в большом документе сложно работать с разметкой.

# Почему выбрали Slate.js?

* поддержка плагинов для гибкой настройки;
* настраиваемая сериализация (в JSON);
* экосистема React;
* неплохая производительность на больших документах.
